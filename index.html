Разделы физики для первокурсников

  • Механика
    • Кинематика

      Прямолинейное равномерное движение:

      • s = vt

      Преобразования:

      • v = s/t
      • t = s/v

      Прямолинейное равноускоренное движение:

      • s = v0t+at2/2
      • s = v+v0/2t
      • v = v0+at
      • v2 = v02+2as
      • a = v-v0/t

      Преобразования:

      • t = (v-v0)/a
      • t = (-v0±√(v02+2as))/a
      • v2 = v02+2as
      • a = (v2-v02)/2s/li>
      • s = (v2-v02)/2a
      • v0 = (v2-2as)/v
      • at = v - v0
      • 2as = v2-v02

      Движение по окружности:

      • aцс = v2/R

      Преобразования:

      • v2 = aцсR
      • R = v2/aцс

      Условные обозначения:

      • v — скорость (м/с)
      • s — расстояние (м)
      • t — время (с)
      • a — ускорение (м/с2)
      • R — радиус (м)
    • Динамика

      Второй закон Ньютона:

      • F = ma

      Преобразования:

      • m = F/a
      • a = F/m

      Закон всемирного тяготения:

      • F = Gm1m2/r2

      Кинетическая энергия:

      • Ek = mv2/2

      Потенциальная энергия упругой деформации:

      • E = kx2/2

      Потенциальная энергия гравитационного поля:

      • Ep = mgh

      Преобразования:

      • m = F/a
      • a = F/m
      • m1 = Fr2/(Gm2)
      • m2 = Fr2/(Gm1)
      • r = √(Gm1m2/F)
      • m = (2Ek)/v
      • v = √((2Ek)/v)
      • k = (2E)/x2
      • x = √(2E/k)
      • m = Ep/(gh)
      • h = Ep/(mg)

      Условные обозначения:

      • F — сила (Н)
      • m — масса (кг)
      • a — ускорение (м/с2)
      • G — гравитационная постоянная
      • m1, m2 — массы взаимодействующих тел (кг)
      • r — расстояние между центрами масс тел (м)
      • v — скорость (м/c, метр в секунду)
      • x — удлинение пружины (м, метр)
      • h — высота (м, метр)
    • Статика

      Условие равновесия:

      • ∑F = 0
      • ∑M = 0

      Условные обозначения:

      • ∑F — сумма всех сил (Н)
      • ∑M — сумма всех моментов сил (Н)
    • Механические колебания и волны

      Гармонические колебания:

      • x = Acos(ωt+φ)

      Гармонические колебания:

      • ω = 2πf

      Период колебаний:

      • T = 1/f

      Преобразования:

      • A = x/cos(ωt+φ)
      • ω = (acos(x/A)-φ)/t
      • t = (acos(x/A)-φ)/ω
      • f = ω/2π

      Условные обозначения:

      • x — смещение (м)
      • A — амплитуда (м)
      • ω — циклическая частота (рад/с)
      • t — время (с)
      • φ — начальная фаза (рад)
      • f — частота (Гц)
      • T — период колебаний (с)
  • Молекулярная физика и термодинамика
    • Основы молекулярно-кинетической теории

      Основное уравнение МКТ:

      • p =(1/3)m0nv2

      Условные обозначения:

      • m0 — масса одной молекулы (г/моль)
      • n — концентрация молекул (м-3)
      • v2 — среднее квадратичное значение скорости молекул(м2/с2)
    • Тепловые явления

      Уравнение состояния идеального газа:

      • pV=nRT

      Закон Бойля-Мариотта (для изотермического процесса):

      • pV=const

      Закон Гей-Люссака (для изобарического процесса):

      • V/T=const

      Закон Шарля (для изохорного процесса):

      • P/T=const

      Преобразования:

      • p = nRT/V
      • V = nRT/p
      • n = pV/(RT)
      • R = pV/(nT)
      • T = pV/(nR)
      • V1 = (p2V2)/p1
      • V1 = (T1V2)/T2
      • p1 = (T1p2)/T2

      Условные обозначения:

      • p — давление (Па)
      • V — объем (м3)
      • n — количество вещества (моль)
      • R — универсальная газовая постоянная
      • T — температура (К)
    • Первый закон термодинамики

      Первый закон термодинамики:

      • ΔU = Q - A

      Преобразования:

      • Q = ΔU+A
      • A = Q-ΔU

      Условные обозначения:

      • ΔU — изменение внутренней энергии (Дж)
      • Q — количество теплоты (Дж)
      • A — работа (Дж)
  • Электродинамика
    • Электростатика

      Закон Кулона:

      • F = keq1q2/r2

      Электрическое поле:

      • E = F/q

      Потенциал электрического поля:

      • φ = W/q

      Потенциальная энергия электрического поля:

      • Ep = qU

        • Преобразования:

        • q1 = (Fr2)/(keq2
        • q2 = (Fr2)/(keq1
        • r = √((keq1q2)/F)
        • F = Eq
        • q = F/E
        • W = φq
        • q = W/φ
        • q = Ep/U
        • U = Ep/q

        Условные обозначения:

        • ke - коэффициент пропорциональности (кулоновская постоянная)
        • q1, q2 - заряды (Кл)
        • r — расстояние между зарядами (м)
        • E — напряженность электрического поля (В/м)
        • F — сила (Н)
        • q — заряд (Кл)
        • φ — потенциал (В)
        • W — работа (Дж)
        • U - напряжение (В, Вольт)
    • Постоянный электрический ток

      Закон Ома для участка цепи:

      • I = U/R

      Работа и мощность электрического тока:

      • A=UIt
      • P=UI

      Преобразования:

      • U = IR
      • R = U/I
      • U = A/(It)
      • I = A/(Ut)
      • t = A/(UI)
      • U = P/I
      • I = P/U

      Условные обозначения:

      • I — сила тока (А)
      • U — напряжение (В)
      • R — сопротивление (Ом)
      • A — работа (Дж)
      • P — мощность (Вт)
      • t — время (с)
    • Магнитное поле

      Сила Лоренца:

      • F=qvBsinα

      Закон Ампера:

      • F=BIlsinα

      Закон Фарадея (электромагнитная индукция):

      • E = -ΔΦ/Δt

      Преобразования:

      • q = F/(vBsinα)
      • v = F/(qBsinα)
      • B = F/(qvsinα)
      • sinα = F/(qvB)
      • B = F/(Ilsinα)
      • I = F/(Blsinα)
      • l = F/(BIsinα)
      • sinα = F/(BIl)
      • ΔΦ = −EΔt
      • Δt = −ΔΦ/E

      Условные обозначения:

      • F — сила Лоренца
      • q — заряд (Кл)
      • v — скорость заряда (А)
      • B — магнитная индукция (Тл)
      • α — угол между направлением движения заряда и магнитным полем
      • I — сила тока (А)
      • l — длина проводника (м)
      • E — ЭДС индукции (тл)
      • Φ — магнитный поток (Вб)
  • Оптика
    • Геометрическая оптика

      Закон отражения света:

      • θi=θr

      Закон отражения света (закон Снеллиуса):

      • n1sin(θ1) = n2sin(θ2)

      Преобразования:

      • n1 = (n2sin(θ2))/sin(θ1)
      • n2 = (n1sin(θ1))/sin(θ2)
      • θ1 = sin-1((n2sin(θ2))/n1)
      • θ2 = sin-1(((n1sin(θ1))/n2)

      Условные обозначения:

      • θi — угол падения
      • θr — угол отражения
      • n1,n2 — показатели преломления сред
      • θ1,θ2 — углы падения и преломления
    • Волновая оптика

      Условие интерференции:

      • dsinθ = mλ

      Условие дифракции:

      • asinθ = mλ

      Преобразования:

      • d = (mλ)/sin(θ)
      • θ = sin-1((mλ)/d)
      • λ = (dsin(θ))/m
      • a = (mλ)/sin(θ)
      • θ = sin-1((mλ)/a)
      • λ = (asin(θ))/m

      Условные обозначения:

      • d — расстояние между щелями (м)
      • θ — угол
      • m — порядок максимума
      • λ — длина волны (м)
      • a — ширина щели (м)
  • Квантовая физика и атомная физика
    • Фотоэффект

      Уравнение Эйнштейна для фотоэффекта:

      • hν = Aвых + Eкин

      Условные обозначения:

      • h — постоянная Планка
      • ν — частота света (Гц)
      • Aвых — работа выхода (Дж)
      • Eкин — кинетическая энергия выбитого электрона (Дж)
    • Строение атома

      Радиус Бора:

      • rn = n2ℏ2/ke2me

      Условные обозначения:

      • n — главный квантовый номер
      • ℏ — приведенная постоянная Планка
      • k — коэффициент пропорциональности
      • e — заряд электрона (Кл)
      • me — масса электрона (а.е.м.)
  • Ядерная физика
    • Энергия связи ядра

      Энергия связи:

      • E=Δmc2

      Преобразования:

      • Δm = E/c2
      • c = √(E/Δm)

      Условные обозначения:

      • E — энергия связи (МэВ)
      • Δm — дефект массы (МэВ/с2)
      • c — скорость света
    • Радиоактивный распад

      Закон радиоактивного распада:

      • N(t) = N0e-λt

      Преобразования:

      • N0 = N(t)/e-λt
      • N0 = N(t)eλt
      • N(t) = N0e-λt
      • λ = -ln(N(t)/N0)/t
      • t = -ln(N(t)/N0)/λ

      Условные обозначения:

      • N(t) — количество ядер в момент времени t
      • N0 — начальное количество ядер
      • λ — постоянная распада (Бк)
      • ln — натуральный логарифм
    • Период полураспада

      Период полураспада:

      • T1/2 = ln(2)/λ

      Преобразования:

      • λ = ln(2)/T1/2
      • λ = -ln(N(t)/N0)/t
      • t = -ln(N(t)/N0)/λ

      Условные обозначения:

      • N(t) — количество ядер в момент времени t
      • N0 — начальное количество ядер
      • λ — постоянная распада (Бк)
      • ln — натуральный логарифм
